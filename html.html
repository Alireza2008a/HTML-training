🚀 HTML چیست؟

HTML مخفف HyperText Markup Language است. این زبان برای ساخت صفحات وب استفاده می‌شود و پایه و اساس تمام وب‌سایت‌های دنیا محسوب می‌شود.

🎯 ویژگی‌های HTML (کلیک کنید)

🏗️ ساختار دهنده
🔗 پیوند دهنده
📱 چندرسانه‌ای
🌐 استاندارد جهانی
♿ دسترسی‌پذیر
🔍 SEO دوست
نکته حرفه‌ای: HTML5 امروزه استاندارد مدرن وب است و قابلیت‌های پیشرفته‌ای مثل Semantic Elements، Canvas، و Web APIs را ارائه می‌دهد.

🔧 مفاهیم کلیدی

🏷️ تگ (Tag)

دستورات HTML که با < و > نوشته می‌شوند

<h1>

📦 المان (Element)

تگ باز + محتوا + تگ بسته

<h1>عنوان</h1>

⚙️ ویژگی (Attribute)

اطلاعات اضافی درون تگ

class="header"

نمونه تعاملی HTML

<h1 class="title">
سلام دنیای HTML!
</h1>

سلام دنیای HTML!

🏗️ ساختار پایه HTML

هر صفحه HTML باید ساختار مشخصی داشته باشد:

<!-- اعلام نوع سند --> <!DOCTYPE html> <html lang="fa" dir="rtl"> <head> <!-- اطلاعات صفحه --> <meta charset="UTF-8"> <title>عنوان صفحه</title> </head> <body> <!-- محتوای قابل مشاهده --> <h1>سلام دنیا!</h1> </body> </html>

توضیح هر بخش:

  • <!DOCTYPE html>: نوع سند HTML5 را مشخص می‌کند
  • <html>: ریشه سند، شامل تمام محتوا
  • <head>: اطلاعات صفحه (غیرقابل مشاهده)
  • <body>: محتوای اصلی صفحه (قابل مشاهده)

🎯 نمونه زنده:

سلام دنیا!

این اولین صفحه HTML من است.

📝 تمرین:

یک صفحه HTML با عنوان "صفحه من" و یک پیام خوشامدگویی بسازید.

📝 تگ‌های متنی

برای نمایش و قالب‌بندی متن از تگ‌های مختلفی استفاده می‌شود:

سرتیترها (Headings):

<h1>سرتیتر اصلی</h1> <h2>سرتیتر فرعی</h2> <h3>سرتیتر سطح سوم</h3> <!-- تا h6 ادامه دارد -->

🎯 نمونه زنده:

سرتیتر اصلی

سرتیتر فرعی

سرتیتر سطح سوم

قالب‌بندی متن:

<p>پاراگراف معمولی</p> <strong>متن پررنگ</strong> <em>متن تاکیدی</em> <br> <!-- شکست خط --> <hr> <!-- خط جداکننده -->

🎯 نمونه زنده:

این یک پاراگراف معمولی است.

این متن پررنگ است و این متن تاکیدی است.

خط بالا جداکننده است.

💡 نکته: از <strong> و <em> به جای <b> و <i> استفاده کنید چون معنای بیشتری دارند.

📋 لیست‌ها

برای نمایش فهرست‌ها از سه نوع لیست استفاده می‌شود:

لیست نامرتب (Unordered List):

<ul> <li>مورد اول</li> <li>مورد دوم</li> <li>مورد سوم</li> </ul>

لیست مرتب (Ordered List):

<ol> <li>قدم اول</li> <li>قدم دوم</li> <li>قدم سوم</li> </ol>

🎯 نمونه زنده:

لیست خرید:

  • نان
  • شیر
  • تخم مرغ

دستور پخت:

  1. مواد را آماده کنید
  2. در تابه بریزید
  3. 15 دقیقه بپزید

لیست تعریف (Definition List):

<dl> <dt>HTML</dt> <dd>زبان نشانه‌گذاری صفحات وب</dd> <dt>CSS</dt> <dd>زبان طراحی صفحات وب</dd> </dl>

📝 تمرین:

یک لیست از علایق خود بسازید و یک لیست مرتب از اولویت‌هایتان.

🔗 لینک‌ها و تصاویر

لینک‌ها (Links):

<!-- لینک به سایت خارجی --> <a href="https://www.google.com" target="_blank">گوگل</a> <!-- لینک به بخش دیگر همین صفحه --> <a href="#section1">برو به بخش 1</a> <!-- لینک ایمیل --> <a href="mailto:test@example.com">ایمیل بفرست</a>

🎯 نمونه زنده:

🔍 جستجو در گوگل

📧 ارسال ایمیل

ویژگی‌های مهم لینک:

  • href: آدرس مقصد
  • target="_blank": باز کردن در تب جدید
  • title: توضیح هنگام hover

تصاویر:

<img src="image.jpg" alt="توضیح تصویر" width="300" height="200">

🎯 نمونه تصویر SVG:

💡 نکته مهم: همیشه ویژگی alt برای تصاویر بنویسید. این برای دسترسی‌پذیری مهم است.

📊 جداول

برای نمایش داده‌های جدولی از تگ‌های table استفاده می‌شود:

<table> <thead> <tr> <th>نام</th> <th>سن</th> <th>شهر</th> </tr> </thead> <tbody> <tr> <td>علی</td> <td>25</td> <td>تهران</td> </tr> </tbody> </table>

🎯 نمونه زنده:

نام سن شهر
علی 25 تهران
سارا 22 اصفهان

اجزای جدول:

  • <table>: کانتینر اصلی جدول
  • <thead>: سربرگ جدول
  • <tbody>: بدنه جدول
  • <tr>: ردیف جدول
  • <th>: سلول سربرگ
  • <td>: سلول داده

📝 تمرین:

جدولی برای نمایش نمرات درس‌هایتان بسازید.

📝 فرم‌ها

فرم‌ها برای دریافت اطلاعات از کاربر استفاده می‌شوند:

<form> <label for="name">نام:</label> <input type="text" id="name" name="name"> <label for="email">ایمیل:</label> <input type="email" id="email" name="email"> <button type="submit">ارسال</button> </form>

🎯 نمونه زنده:

✅ فرم با موفقیت ارسال شد!

انواع input:

  • text: متن ساده
  • email: آدرس ایمیل
  • password: رمز عبور
  • number: عدد
  • checkbox: چک باکس
  • radio: دکمه رادیویی
💡 نکته: همیشه از label برای فیلدهای فرم استفاده کنید تا دسترسی‌پذیری بهتر شود.

🎯 تگ‌های معنایی HTML5

HTML5 تگ‌های جدیدی معرفی کرد که معنای بیشتری به محتوا می‌دهند:

<header> <nav>منوی ناوبری</nav> </header> <main> <article> <section>بخش مقاله</section> </article> <aside>نوار کناری</aside> </main> <footer>پاورقی صفحه</footer>

🎯 نمونه ساختار معنایی:

Header - سربرگ
Main Content

محتوای اصلی صفحه اینجا قرار می‌گیرد.

Aside

نوار کناری

Footer - پاورقی

مزایای تگ‌های معنایی:

  • 🔍 SEO بهتر: موتورهای جستجو بهتر می‌فهمند
  • ♿ دسترسی‌پذیری: برای افراد نابینا مفیدتر است
  • 🧹 کد تمیزتر: ساختار واضح‌تری دارد
  • 🤖 خوانایی ماشین: برای ربات‌ها قابل فهم‌تر است

🎉 تبریک! دوره را تمام کردید

حالا می‌تونید یک صفحه کامل HTML بسازید. تمرین نهایی: یک صفحه شخصی با تمام مفاهیم یادگرفته شده بسازید.

مثال‌های کاربردی بیشتر:

1️⃣ صفحه رستوران:

<header> <h1>رستوران سنتی</h1> <nav> <a href="#menu">منو</a> <a href="#contact">تماس</a> </nav> </header> <section id="menu"> <h2>منوی غذا</h2> <table> <tr> <th>غذا</th> <th>قیمت</th> </tr> <tr> <td>چلو کباب</td> <td>150,000 تومان</td> </tr> </table> </section>

2️⃣ فرم ثبت‌نام:

<form> <fieldset> <legend>اطلاعات شخصی</legend> <label for="fname">نام:</label> <input type="text" id="fname" required> <label for="age">سن:</label> <input type="number" id="age" min="18" max="100"> <label>جنسیت:</label> <input type="radio" name="gender" value="male"> مرد <input type="radio" name="gender" value="female"> زن </fieldset> <button type="submit">ثبت‌نام</button> </form>

3️⃣ مقاله وبلاگ:

<article> <header> <h1>آموزش HTML</h1> <time datetime="2024-01-15">15 ژانویه 2024</time> <address>نویسنده: <a href="mailto:author@example.com">علی احمدی</a></address> </header> <section> <h2>مقدمه</h2> <p>HTML زبان اصلی وب است...</p> <blockquote cite="https://www.w3.org"> <p>وب برای همه ساخته شده است.</p> <footer>— <cite>تیم برنرز لی</cite></footer> </blockquote> </section> <footer> <p>برچسب‌ها: <a href="#html">#HTML</a> <a href="#آموزش">#آموزش</a> </p> </footer> </article>
